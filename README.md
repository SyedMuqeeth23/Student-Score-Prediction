# 📘 Student Score Prediction App

## 🚀 Overview

This project is a Machine Learning web application built using **Streamlit** to predict student scores based on input features.

---

## 🧠 Features

* 📊 Predict student scores using ML model
* ⚡ Real-time predictions
* 🎯 User-friendly Streamlit interface
* 🧹 Data preprocessing and model training

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas & NumPy
* Scikit-learn
* Streamlit

---

## ⚙️ Installation

1. Clone the repository:

```id="6wx1dc"
git clone https://github.com/SyedMuqeeth23/Student-Score-Prediction.git
```

2. Navigate to the folder:

```id="5o1g1o"
cd Student-Score-Prediction
```

3. Install dependencies:

```id="3lh6lp"
pip install -r requirements.txt
```

---

## ▶️ Run the App

```id="a7wy79"
streamlit run app.py
```
